Description
-----------
This module create a block for helping users to : 
   * add the website to their browser's bookmarks.
   * make the website as their browser's homepage.

Installation
------------
To install, copy the addtofavorites directory and all its contents to your modules
directory.

Configuration
-------------
To enable this module, visit administer -> modules, and enable addtofavorites.
To configure the different options, and the text to display visit administer -> settings --> addtofavorites
